Github+Hexo:个人博客

环境及版本

  • window 10
  • hexo 3.4.0

安装 git 和 Node.js

  1. 安装 Node.js
  2. 安装 Git for windows

  3. 修改 npm 镜像源为淘宝镜像源。

    1. 临时方式

    2. 持久方式

      • 配置后可通过npm config get registry来验证是否成功。

安装 hexo，hexo-deployer-git

  1. 安装 hexo。

    如果提示INFO Start blogging with Hexo!则表示初始化成功。

  2. 进入blog_on_hexo并安装依赖。

  3. 为了将博客部署到 github 需要安装 hexo-deployer-git。

    否则后面使用hexo deploy时会报出ERROR Deployer not found: git的错误。

    npm install hexo-deployer-git

    • 会把hexo-deployer-git包安装到node_modules目录中
    • 不会修改package.json
    • 之后运行npm install命令时，不会自动安装hexo-deployer-git
      npm install --save
    • 会把hexo-deployer-git包安装到node_modules目录中
    • 会在package.json的dependencies属性下添加hexo-deployer-git
    • 之后运行npm install命令时，会自动安装hexo-deployer-git到node_modules目录中
    • 之后运行npm install --production或者注明NODE_ENV变量值为production时，会自动安装hexo-deployer-git到node_modules目录中
      npm install --save-dev
    • 会把hexo-deployer-git包安装到node_modules目录中
    • 会在package.json的devDependencies属性下添加hexo-deployer-git
    • 之后运行npm install命令时，会自动安装hexo-deployer-git到node_modules目录中
    • 之后运行npm install --production或者注明NODE_ENV变量值为production时，不会自动安装hexo-deployer-git到node_modules目录中

  1. 在 github 上新建一个名为username.github.io的仓库，username为自己的 github 用户名，对我来说就是 RobertLexis。
  2. 编辑blog_on_hexo\_config.yml。
