目标定位 vs 目标检测

2018-08-26

计算机视觉领域的不同任务：


主要任务类别

  • Classification
    图像中有一个主体目标，且该张图片被标记为该主体目标的类别。
    • one main object per image
    • one class label per image

one-hot vector
\[
y = \left[
\begin{matrix}
c_1 \\\\
c_2 \\\\
c_3
\end{matrix}
\right]
\]

  • Classification + Localization
    图像中有一个主体目标，且该张图片被标记为该主体的类别和位置。
    • one main object per image
    • one class label per image

\[
y = \left[
\begin{matrix}
p_c \\\\
b_x \\\\
b_y \\\\
b_w \\\\
b_h \\\\
c_1 \\\\
c_2 \\\\
c_3
\end{matrix}
\right]
\]

其中 \( p_c \) 标识图像中是否出现目标，\( (b_x, b_y, b_w, b_h) \) 标识若图像中目标存在则图像中目标的位置，\( (c_1, c_2, c_3) \) 标识若图像中目标存在则图像中目标的类别（此处以三类目标为例）。

  • Object Detection
    图像中有属于多个类别的多个目标
    • multi objects per image
    • multi classes per image

推广任务类别

  • Multi label classification
    图像中有一个主体目标，且该张图片被标记为该主体目标的类别，但该主体目标可能同时属于多个类别。
    • one main object per image
    • multi class labels per image

\[
y = \left[
\begin{matrix}
c_1 \\\\
c_2 \\\\
c_3
\end{matrix}
\right]
\]
\( y \) 根据图像中主体目标所属类别的个数不一定是 one-hot 的。

  • Multi objects Classification + Localization
    • n objects per image
    • m classes per image
    • \( n_{c_1} + n_{c_2} + n_{c_3} + … + n_{c_m} = n \)
    • \( n, m, n_{c_i} \) 对于一个具体任务是一组确定的值。

\[
y\_category = \left[
\begin{matrix}
p_c \\\\
c_1 \\\\
c_2 \\\\
c_{3, 1} \\\\
c_{3, 2}
\end{matrix}
\right]
y\_location = \left[
\begin{matrix}
\\\\
b_{x, 1} \\\\
b_{y, 1} \\\\
b_{w, 1} \\\\
b_{h, 1} \\\\
b_{x, 2} \\\\
b_{y, 2} \\\\
b_{w, 2} \\\\
b_{h, 2} \\\\
b_{x, (3, 1)} \\\\
b_{y, (3, 1)} \\\\
b_{w, (3, 1)} \\\\
b_{h, (3, 1)} \\\\
b_{x, (3, 2)} \\\\
b_{y, (3, 2)} \\\\
b_{w, (3, 2)} \\\\
b_{h, (3, 2)}
\end{matrix}
\right]
\]
