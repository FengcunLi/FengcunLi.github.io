Github+Hexo:个人博客

环境及版本

  • window 10
  • hexo 3.4.0

安装 git 和 Node.js

  1. 安装 Node.js
  2. 安装 Git for windows

  3. 修改 npm 镜像源为淘宝镜像源。

    1. 临时方式

    2. 持久方式

      • 配置后可通过npm config get registry来验证是否成功。

安装 hexo，hexo-deployer-git

  1. 安装 hexo。

  2. 新建目录 blog_on_hexo，不要进入blog_on_hexo。

如果提示INFO Start blogging with Hexo!则表示初始化成功。

  1. 进入blog_on_hexo并安装依赖。

  2. 为了将博客部署到 github 需要安装 hexo-deployer-git
